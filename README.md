# cna-template

A starter template for building applications on top of **CNA**, a C++
reimplementation of the XNA 4.0 game framework programming model, built on
SDL3 with a pluggable graphics backend layer.

CNA lives in a sibling repository (`../cna`, see the
[CNA project](https://github.com/openeggbert/cna) for details).

## Status

Work in progress — full build instructions, backend/platform coverage, and a
guide for porting existing XNA 4.0 C# games are being added on the `develop`
branch.
